# MovePulseSport — Website

Dies ist eine statische Website-Vorlage für MovePulseSport, inspiriert von freegamemgmt.com.

Wie nutzen
1. Lege die Dateien im Repo-Root ab oder in `docs/`, wenn du GitHub Pages aus `docs/` nutzen willst.
2. Passe `index.html` (Texte, Logos, Screenshots) an euer Produkt an.
3. Passe Farben in `css/styles.css` an.
4. Lokal ansehen: öffne `index.html` im Browser oder nutze einen lokalen Dev-Server:
   - Python 3: `python -m http.server 8000` und öffne http://localhost:8000
5. Deploy:
   - GitHub Pages: aktiviere Pages vom Branch `main` Root oder `docs/`.
   - Netlify/Vercel: Verbinde das Repo — kein Build nötig für statische Dateien.

Möchtest du, dass ich die Dateien ins Repo pushe? Gib mir:
- den gewünschten Branch-Namen (z. B. `feature/site`) und
- ob die Dateien im Repo-Root oder in `docs/` landen sollen.
